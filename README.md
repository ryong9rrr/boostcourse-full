# 🎯 요구사항

[] 할 일을 입력하면 Todo 칸에 할 일이 추가된다.

[] Todo 에서 > 버튼을 누르면 Doing으로 할 일이 옮겨 간다.

[] Doing 에서 > 버튼을 누르면 Done으로 할 일이 옮겨 간다.

[] Done 에서 x 버튼을 누르면 삭제된다.

## ✅ Step 1

기본 요구사항을 구현한다.

## ✅ Step 2

로컬스토리지를 이용하여 새로고침해도 데이터가 남아있도록 한다.
